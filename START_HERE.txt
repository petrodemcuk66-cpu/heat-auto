START HERE

1. Розпакуй архів.
2. Відкрий папку у VS Code.
3. Перевір, що у .env / .env.local залишились твої Neon і KeyCRM дані.
4. У терміналі введи:

npm install
npx prisma generate
npx prisma migrate deploy
npm run db:seed
npm run dev

5. Відкрий:

http://localhost:3000

6. Перед хостингом:

npm run ready
