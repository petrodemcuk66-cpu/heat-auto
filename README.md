# HEAT AUTO — USA / Europe final

Оновлена версія сайту HEAT AUTO після правок клієнта.

## Що змінено

- прибрано Канаду з публічних текстів, Hero, напрямків, авто і прикладів;
- залишено тільки США та Європу;
- додано телефон: `+38 (063) 654-06-00`;
- додано Telegram: `@HeatAuto`;
- у метриках вказано економію `30–40%`;
- Telegram і телефон підключені через `src/data/site.ts`, `Footer`, `Header`, `FloatingActions`, `Contacts`;
- `.env` та `.env.local` збережені, оновлені тільки публічні `NEXT_PUBLIC_PHONE` і `NEXT_PUBLIC_TELEGRAM`.

## Запуск

```bash

npx prisma generate
npx prisma migrate deploy

npm run dev
```

## Fixed mobile menu overlay

Виправлено баг, коли після натискання на hamburger-кнопку зʼявлявся хрестик, але саме меню не було видно.

Що змінено:

- меню винесено як окремий overlay поруч із header;
- header має `z-[100]`;
- mobile menu має `z-[90]`;
- меню має `fixed inset-x-0 bottom-0 top-20`;
- фон меню тепер реальний `bg-[#050507]`, а не прозорий;
- скрол сторінки блокується через `body` і `html`;
- всередині меню є швидкий доступ: телефон, Telegram, калькулятор, авто, заявка.
